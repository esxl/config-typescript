## Install

```sh
npm install --save-dev @esxl/config-typescript
```

## Use

Use the [`extends`](https://www.typescriptlang.org/tsconfig#extends) configuration property in your _tsconfig.json_:

```json
{
  "extends": "@esxl/config-typescript"
}
```
